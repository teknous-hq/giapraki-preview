# giapraki — brand preview

Δύο αισθητικές κατευθύνσεις (A-warm + B-clean) για το giapraki — ίδιο logo, ίδιο cart engine, πλήρες flow:
**αρχική → εστιατόρια → μενού → καλάθι → παραγγελία → tracking**.

- **A — Warm:** coral hero + saffron accents + food photos. Ζεστή, παιγνιώδης. _Προτιμώμενο._
- **B — Clean:** ink + coral accent + hairline borders. Editorial, premium magazine.

Άνοιξε το `index.html` ως selector. Cart κρατιέται σε `localStorage` και είναι κοινό μεταξύ A/B.

Σχέδιο πρότασης — όχι επίσημο site.
